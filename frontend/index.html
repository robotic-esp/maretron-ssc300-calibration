Maretron SSC300 Calibration

This tool allows for the SSC300 compass to be calibrated. To calibrate the heading, click "Start Calibration" under the Heading Calibration section. To set the installation offset, enter the current device heading in degrees to one decimal place (e.g, 121.7Â°) and click "Start Calibration."
Note: For this tool to work, the NMEA 2000 network containing the SSC300 must be connected with the Actisense NGX-1 (or NGT-1) bridge.
Note: the SSC300 must already be installed in the desired location before calibration is performed. Calibration requires making 3 complete turns, so your boat must be in the water and in a clear area to perform the calibration.
